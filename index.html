Privacy Policy

Last Updated: April 28, 2026

Introduction

STEPN Batch Buyer ("the Extension") is a browser extension designed to enhance the user experience on the STEPN platform by providing batch purchasing capabilities. This Privacy Policy explains what information the Extension collects, how it is used, and how it is protected.

Key Principle: The Extension does not collect or transmit any personal information to external servers owned by the developer. All data is stored locally on your device, with limited exceptions explicitly described below.

Information We Collect and How We Use It

1. Session ID

The Extension intercepts the Session ID from API requests made by the STEPN website (m.stepn.com) to the STEPN API. This Session ID is:

  • Source: Extracted from URL parameters in requests to api.stepn.com
  • Purpose: Used to authenticate API calls to STEPN's official API for purchasing items, querying inventory, and managing orders
  • Storage: Stored locally in your browser using chrome.storage.local
  • Transmission: Only sent to STEPN's official API servers, never to third-party servers owned by the developer

2. Cart and Preference Data

The Extension stores the following data locally on your device:

Data Type Purpose Storage Location
Cart items (orderID, OTD, price, item details) Batch purchasing functionality chrome.storage.local
User settings (buy interval, preferences) Customizing user experience chrome.storage.local
Purchase logs User's purchase history reference chrome.storage.local
Premium status Tracking premium activation chrome.storage.local

3. External API Communications

The Extension communicates with the following external services:

STEPN Official API (api.stepn.com)

  • Purpose: Core functionality - purchasing items, querying inventory, managing orders
  • Data Sent: Session ID, order IDs, price information
  • Note: This is the official STEPN API. Your interaction with this API is subject to STEPN's own privacy policy

Solana Blockchain RPC Nodes

  • Purpose: Verifying premium payment transactions on the Solana blockchain
  • Data Sent: Transaction signature (provided by user)
  • Services Used: api.mainnet-beta.solana.com, rpc.ankr.com, solana-rpc.publicnode.com
  • Note: These are public blockchain nodes. No personal data is collected beyond the transaction signature you provide

Supabase

  • Purpose: Recording used transaction signatures to prevent reuse
  • Data Sent: Transaction signature (only when activating premium)
  • Service: gqdbewcwctyozrdpuodn.supabase.co
  • Note: Only transaction signatures are stored, no personal data. This is to prevent the same transaction from being used to activate premium on multiple devices

Price Data APIs

  • Purpose: Displaying current token prices
  • Services: api.coingecko.com, api.dexscreener.com
  • Data Sent: None (public price queries only

Information We Do NOT Collect

The Extension does does NOT:

  • Collect personally identifiable information (name, email, address, etc.)
  • Collect browsing history or data from websites other than STEPN
  • Track your activity across different websites
  • Collect keystrokes or sensitive input data
  • Upload your local data to developer-owned servers
  • Sell or share your data with third parties for advertising or marketing purposes

Data Storage and Security

Local Storage

All user data is stored locally on your device using Chrome's chrome.storage.local API, which is:

  • Isolated to this extension only
  • Not accessible by other extensions or websites
  • Encrypted at rest by Chrome's security mechanisms

Data Retention

  • All local data is retained until you:
    • Clear the extension's storage via Chrome settings
    • Uninstall the extension
    • Manually clear data within the extension
  • Purchase logs are automatically limited to the last 200 entries

Permissions Justification

The Extension requests the following permissions, each with a specific purpose:

webRequest

Purpose: Intercept API requests to extract the Session ID from STEPN's API calls. This is necessary to authenticate with STEPN's API for purchasing functionality.

storage

Purpose: Store cart items, user settings, purchase logs, and session data locally on your device.

activeTab

Purpose: Communicate with the currently active tab when on STEPN's website to inject UI elements and interact with the page.

Host Permissions

The Extension requests access to specific domains only:

  • api.stepn.com - Core STEPN API functionality
  • m.stepn.com - STEPN web application
  • Solana RPC nodes - Payment verification
  • Supabase - Transaction signature tracking
  • Price APIs - Market data display
